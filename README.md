# Gerador de Senhas Seguras

Gerador de senhas com interface moderna estilo glassmorphism, desenvolvido com HTML5, CSS3 e JavaScript puro.

## Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/password-generator.git
   cd password-generator
   ```

2. Abra o arquivo `index.html` no navegador ou use um servidor local:
   ```bash
   python -m http.server 8000
   ```

## Tecnologias

- HTML5
- CSS3 (Custom Properties, Flexbox, Backdrop-filter, Animações)
- JavaScript (ES6+)
